# parking-manager â€” Quick Start

This repository contains a Node.js backend (`backend/`) and a Flutter frontend (`frontend/`).

Prerequisites:
- Node.js & npm (for the backend) 
- Flutter setup (for the frontend)
- MySQL server(Xampp)

Run the project (PowerShell)

1) Prepare the database (MySQL)
   -Open Xampp and start the apache and MySQL
2) Start backend (new terminal)

```powershell
cd backend (goto the backend folder of this project in terminal or cmd or powershell)
npm install
npm start
```

3) Start frontend (separate terminal)

```powershell
cd frontend (goto the frontend folder of this project in terminal or cmd or powershell)
flutter pub get
flutter run (to run in a browser press any of the browser showing in the list if any emulator/android devices are not connected)
```
